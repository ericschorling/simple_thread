# Simple Thread Reimbursment Program
This program accepts an array of projects. Each project should be submitted in the ["9/1/15", "9/2/15","H" format. Indicating the first day [0], last day[1], and cost of the city[2]. Input will be validated.

In the root directory, you can run:

### `node reimbursement.js`

This will run the four preselected sets. You will need to modify reimbursement.js to add in different sets to test.
